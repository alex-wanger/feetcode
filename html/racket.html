History of Racket

Racket is a general-purpose, multi-paradigm programming language. It is a modern dialect of Lisp and a descendant of Scheme. Racket was created to support language development, education, and research in computer science.

It was designed by Matthias Felleisen and his team as part of the PLT (Programming Language Team). The project began in the mid-1990s and evolved into one of the most flexible and educationally-used functional languages today.

Racket is known for its powerful macro system and language extensibility, allowing programmers to create custom languages. It is widely used in academic settings as well as practical software development.

Basics of Racket

Print Statements

In Racket, expressions are written using parentheses. Functions are placed before the values they act on (prefix notation).

Example: (print "Hello World")

Output: Hello World

Variables and Functions

To define a function, use the keyword define followed by a name and input.

Example: (define (square x) (* x x))

Usage: (square 5)

Output: 25

Conditional Logic

Racket supports two main forms of conditionals: if (simple) and cond (multiple branches).

Example (if): (if (> 3 2) "Yes" "No")

Output: Yes


Example (cond):

(define x 7)

(cond

  [(> x 10) "Too big"]

  [(< x 5) "Too small"]

  [else "Just right"])

Usage: Evaluate with x set to 7

Output: Just right

Lists

Use the list function to create a list:

Example: (define mylist (list 1 2 3 4))

Usage: mylist

Output: (1 2 3 4)


To get the first item in a list, use car or first.

Usage: (car mylist) or (first mylist)

Output: 1


To get the rest of the list (without the first item), use cdr or rest.

Usage: (cdr mylist) or (rest mylist)

Output: (2 3 4)


You can add elements to the front using cons:

Example: (cons 0 mylist)

Output: (0 1 2 3 4)

Lambda Expressions

A lambda expression creates an anonymous (temporary) function.

Example: ((lambda (x) (+ x 1)) 4)

Output: 5

Recursion

Recursion is when a function calls itself. You must define a base case (when to stop) and a recursive case (the repeat).

Example:

(define (factorial n)

  (if (= n 0)

    1

    (* n (factorial (- n 1)))))

Usage: (factorial 4)

Output: 24


Explanation: This recursive function multiplies down to 1. The base case is when n = 0, and it returns 1. Otherwise, it keeps calling itself with a smaller number. It multiplies 4 × 3 × 2 × 1.
