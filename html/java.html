History of Java

Java was developed at Sun Microsystems, Inc. by a team led by James Gosling from 1991 to 1995. Originally called Oak, the language was designed to be portable, secure, and robust. Its primary goal was to enable the concept of "write once, run anywhere" (WORA), allowing compiled Java code to run on any device equipped with a Java Virtual Machine (JVM) without needing recompilation.

Since its release, Java has grown into one of the most widely-used programming languages in the world. It has been essential in enterprise applications, Android app development, web servers, and large-scale systems. Java's syntax is heavily influenced by C and C++, but it introduced automatic memory management (garbage collection) and a strong emphasis on security and portability. Java’s ecosystem includes an extensive standard library and vibrant open-source community.

Basics of Java

Main Class

Every Java program starts with a class and a main method, which is the program’s entry point. Java syntax is similar to C/C++ with a clear, structured format using semicolons to end statements, curly braces {} to define code blocks, and strong typing for variables and methods.

Example:

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

Output: Hello World

Note: System.out.println() prints the text and moves to a new line, whereas System.out.print() prints text but stays on the same line.

Variables and Data Types

Java is a statically typed language, so every variable must have a declared type.

Type Description Example
String Stores text data inside double quotes String name = "Java";
int Stores integers (whole numbers) int age = 25;
float Stores decimal numbers (single precision) float price = 9.99f;
double Stores decimal numbers (double precision) double pi = 3.14159;
char Stores a single character in single quotes char letter = 'A';
boolean Stores true or false boolean isActive = true;

Variable assignment syntax: type variableName = value;

Arrays

Arrays hold multiple values of the same type in a fixed-size structure.


Example:

int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[2]);

Output: 3

Note: Array indices start at 0.

Loops

For Loop

A for loop repeats a block of code a specific number of times. It has three parts inside the parentheses:

  • Initialization: sets a starting point (e.g., int i = 0;)
  • Condition: loop continues while this is true (e.g., i < 3)
  • Update: runs after each iteration (e.g., i++ increases i by 1)

This structure controls how many times the loop runs.

Example:

for (int i = 0; i < 3; i++) {
  System.out.println(i);
}

Output:
0
1
2

While Loop

A while loop keeps running as long as its condition remains true. You need to initialize variables before the loop, and update them inside the loop to avoid infinite loops.

Example:

int i = 0;
while (i < 3) {
  System.out.println(i);
  i++; // increase i to eventually stop the loop
}

Output:
0
1
2

Object-Oriented Programming (OOP)

Classes and Objects

A class is a blueprint defining the structure (fields) and behavior (methods) of objects. An object is an instance of a class representing something specific.

Example:

public class Student {
  String name;
  int age;

  void introduce() {
    System.out.println("Hi, I'm " + name + ", and I'm " + age + " years old.");
  }
}

Usage:

Student sigma = new Student();
sigma.name = "Sigma";
sigma.age = 17;
sigma.introduce();

Output:

Hi, I'm Sigma, and I'm 17 years old.

Fields (Instance Variables)

Fields hold data for each object. For example, name and age store each Student's information.

Note: Use private fields with getters/setters to protect data (encapsulation).

Methods

Methods define actions objects can perform. They can take parameters and return values.

Example:

public class Calculator {
  int add(int a, int b) {
    return a + b;
  }
}

Usage:

Calculator calc = new Calculator();
int sum = calc.add(5, 3);

Output:

8

Constructors

A constructor initializes new objects. It has the same name as the class and no return type.

Example:

public class Student {
  String name;
  int age;

  public Student(String name, int age) {
    this.name = name;
    this.age = age;
  }
}

Usage:

Student sigma = new Student("Sigma", 17);

Inheritance

Inheritance lets a class reuse fields and methods from another class.

Example:

public class Person {
  void greet() { System.out.println("Hello!"); }
}

public class Student extends Person {
  void study() { System.out.println("Studying hard!"); }
}

Usage:

Student sigma = new Student();
sigma.greet();
sigma.study();

Output:

Hello!
Studying hard!

Abstraction

Abstraction hides complex details and shows only essential features. Java uses abstract classes and interfaces for this.

Example abstract class:

public abstract class Vehicle {
  abstract void start();

  void stop() {
    System.out.println("Vehicle stopped.");
  }
}

Example subclass:

public class Car extends Vehicle {
  @Override
  void start() {
    System.out.println("Car started.");
  }
}
